# CyberQuest

CyberQuest is a 2D educational platformer developed in Unity that teaches basic cybersecurity concepts to children aged 12–15 through interactive gameplay and puzzles.  
The project was created as part of the *Software Engineering Project 1* course at Tampere University, in collaboration with university research team acting as the customer.

The goal of the project was to design an engaging game that introduces cybersecurity awareness in a way that is easy to understand.

## Project Overview

CyberQuest is built using the Unity game engine (2D) and programmed in C#. The game consists of five unique levels, each centered around a specific cybersecurity theme. Gameplay varies from classic platforming mechanics to puzzle-based challenges.

The project features:
- 5 unique levels ranging from platforming to puzzles
- Intro, menu, settings, pause controls 
- Custom-made character designs, animations, and music created specifically for the game  

## Learning Objectives

The game aims to teach players:
- Basic password security principles  
- Phishing threat awareness 
- Same-Origin Policy (SOP)

## How to Play

- **Supported platforms:** Windows, Linux, macOS  
- A **Windows build** is available in this repository as a ZIP file.

### Running the game (Windows)

1. Download the ZIP file from the repository  
2. Extract the contents  
3. Run `CyberQuest PROD.exe`
